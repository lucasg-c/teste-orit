## Descrição

Repositório de teste de uma aplicação CRUD feita com [Nest](https://github.com/nestjs/nest).

## Instalação

```bash
$ npm install
```

## Executando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Testes
Foram realizados testes E2E do módulo `customers`.
```bash
# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
